# interpreterpp

This project is a custom-built interpreter developed as part of the Programming Languages and Compilers course. The goal of the project was to design and implement an interpreter for a custom programming language, applying principles of lexical analysis, parsing, and runtime execution.

The interpreter processes source code written in a defined syntax, performs lexical and syntactic analysis, builds an abstract representation of the program, and executes the instructions accordingly.

🔷 Key Features

Lexical analysis (tokenization of source code)

Syntax parsing and grammar validation

Execution of statements and expressions

Variable declaration and assignment handling

Support for control structures (e.g., conditions, loops)

Expression evaluation

Error detection and reporting during parsing and execution
